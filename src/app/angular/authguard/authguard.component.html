Introduction

This is used to restrict the usage of admin panel by the user.

ng g g auth

  • This will create a file named auth.gaurd.ts
  • It will check in if statement that whether data in (res) is empty or not.
  • If data is not empty than console->application->sessionStorage the key and value will be true.
  • And through navigate we can send it to another page.

The key and value which is set in login.component.ts is checked here. If the data is undefined it will redirect to home page and return false.


When login is successfull than it will redirect navigate to admin module.


Create a module named log which will which will be bridge between app module and auth.module.

  • 1 auth.module.ts

    In auth.module.ts we need to import logModule.

  • 2 log.module.ts

    In log.module.ts we need to import logincomponent. It is necessary because through login form we will be able to enter admin panel.

  • 3 app.module.ts

    In app.module.ts we need to import both LogModule and AdminModule.

  • 4 admin-routing.module.ts

    By activating the auth-gaurd without login successfully anyone will not able to visit admin pages.
